# mobile-market

> 项目演示地址：https://lxw-peter.github.io/market/#/main
>
> 本项目开发环境由vue-cli3搭建
>
> 技术栈： Vue-Router axios  Vant ...  

### Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
